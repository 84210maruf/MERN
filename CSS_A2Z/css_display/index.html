Display:block;

Div, P, H1-H6, ul, ol, li, from, section, header, footer ETC.. elements are by-defult Block element

Note : A block-level element ALWAYS starts on a new line and takes up the full width available and customize width, height, padding, margin


Display:inline;

button, span, strong, input, b(bold), a(anchor), em, i(italic),map, br, select, img, small, date, time, lebel ETC..elements are by-defult Block element

Note : An inline element DOES NOT start on a new line and only takes up as much width as necessary. Height, padding[top,bottom], margin[top,bottom] not working... Solution is customize Display: properties As Display:inline-block;
